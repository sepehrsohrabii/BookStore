# BookStore
This a simple store with HTML, CSS, JS, Django(for backend).

<b>Features:</b>
- RTL
- Bootstrap responsive
- 

<b>Steps before run project:</b></br>
1- Create virtualenv:</br>
    &nbsp;&nbsp;&nbsp;&nbsp;$ python3 -m venv venv</br>
2- Activate venv in /venv/scripts folder use this code:</br>
    &nbsp;&nbsp;&nbsp;&nbsp;$ . activate</br>
3- Install requirements.txt (Run this code in where the requirements.txt is.):</br>
    &nbsp;&nbsp;&nbsp;&nbsp;$ pip install -r requirements.txt</br>
5- Create SuperUser by running this code in the main folder:
    &nbsp;&nbsp;&nbsp;&nbsp;$ python manage.py createsuperuser
4- Run project by running this code in the main folder:</br>
    &nbsp;&nbsp;&nbsp;&nbsp;$ python manage.py runserver</br>
